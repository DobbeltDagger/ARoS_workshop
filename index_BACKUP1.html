Brief history of Javascript

Netscape Navigator '93: Approaching Sun Microsystems: Can we use Java on the net? Brendan Eich tried to embed the Scheme Programming Language into the browser, called it Mocha, Livescript and finally javascript (to resemble Java, which was new and popular). Netscape wanted to compete on features with the free Internet Explorer.
In 1995 javascript was launched in Netscape Navigator 2. Microsoft then reverse engineered javascript and named it Jscript. Netscape went to ECMA (European Computer Manufacturers Association) and a new language standard called Ecmascript was launched in 1997. In 1999 Netscape was sold to AOL. Netscape Navigator was made open source (from which Firefox was made). In 2000 the dot.com bubble burst. Macromedia Actionscript was inspired by Ecmascript 4, which was too feature rich for browser vendors to implement. In 2005 we got AJAX. Now we inject data into pages without reload. Jquery is growing big. In 2008 Google invents chrome. V8 engine compiles the code multiple times while running it: The JIT compiler. This is faster than interpreted language. Node.js is based on chrome's V8 engine, and now JS is also for servers and desktop apps. Fullstack.
An even briefer history ...

- The 90s web was static: No dynamic code on pages
- Web 2.0 and social media made javascript ubiquitos
- Javascript is now fullstack. It can be used to make servers and desktop applications
