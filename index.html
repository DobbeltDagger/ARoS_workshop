Brief history of Javascript ...

Ecmascript ... Netscape Navigator ... '93 ... Approching Sun Microsystems -> Can we use Java on the net? Applets? Brendan Eich made the first solution! (The Scheme Programming Language). The name of this language is Mocha! - Changed to Livescript when launched ... Then later named javascript (because java was famous) Netscape wanted to have this awesome lang in their browser, so that it could still compete with IE, that was a free browser. Netscape Navigator was a paid browser ... In 1995 javascript was launched in Netscape Navigator 2! Microsoft then reverse engineered javscript, and named it Jscript! Netscape goes to ECMA (who are responsible for standards ...) -> a new language standard called Ecmascript is presented to the World Ecmascript 1 was launched in 1997 1999 Netscape was sold to AOL ... Netscape Navigator was made open source ... which in turn became Firefox!! 2000 dot.com bubble burst!! Macromedia Actionscript was actually inspired by Ecmascript 4 (which had so many new features that browsers couldn't implement them) 2005 AJAX (Asynchronous Javascript and HTML). Now we can inject data into the page without page reload. Jquery is big here! 2008 Google invents chrome. V8 engine compiles code, so that it is both compiled and interpeted. The jit compiler! (Just in time compiler). This is much faster than just a interpreted language. The compiler makes an.exe file on you machine, and can run the code faster! Node.js is a version of chrome's V8 engine... Now JS is backend also - FULLSTACK! 2009 Ecmascript 5 2015 Ecmascript 6 2016 ES7 .. and from now on it's yearly versions!!! ...
